# SMS Spam Classification

## Capstone Project for Capstone project for [HarvardX PH125.9x, Data Science: Capstone](https://www.edx.org/certificates/professional-certificate/harvardx-data-science)

## Dataset

The dataset is included in the `data` directory of this repo. In addition, it
can be found in the [UCI Machine Learning Repository](https://archive.ics.uci.edu/dataset/228/sms+spam+collection)

## Files

The repo contains the final report as a PDF, the R Markdown file used to generate
the report, and the R script. The R script contains all the same code but makes
use of `parallel` and so it runs a bit faster. Recommend running that over
rebuilding the report.
